fcjz,

目前我在互联网上唯一长期使用的身份。

凡尘芥之，一个并没有什么典故的名字，但包含了我的现在和未来，“凡尘”有我现在的名字，“芥之”则会成为未来的常用名，所以也就诞生了两层含义:

    现实中的我，包括现在和未来
    尘世中的草芥，平凡且自由

国外网站大多数取自这个名字的缩写，也就是fcjz，国内很少有账号。

没有用那些流行的风格，比如各种图标来装饰自己的GitHub readme，来让它显得更酷，但，这个仓库已经表达了所有我想要表达的东西。

欢迎访问花园，欢迎邮件交友，认识你是我的荣幸。
